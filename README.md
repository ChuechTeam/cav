# La CAV

## Prérequis

- Java 25 dans le PATH
- Python 3

## Pour lancer le projet


### Lancer les services
Windows : `python ./run.py discovery service`

Linux : `./run.py discovery service`

### Lancer le client

Windows : `python ./run.py client`

Linux : `./run.py client`
